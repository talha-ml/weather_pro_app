## 🚀 Key Features
- **Auto-Location Detection:** Automatically fetches weather for your current city using GPS.
- **Real-time Weather:** Detailed current weather including Temperature, Humidity, Wind Speed, and "Feels Like".
- **Dynamic Backgrounds:** The app's background changes color based on the current weather conditions (Sunny, Cloudy, Rainy, etc.).
- **Lottie Animations:** Beautifully animated weather icons for an immersive experience.
- **Hourly Temperature Chart:** A professional line chart showing temperature trends for the next 24 hours.
- **5-Day Forecast:** Daily weather predictions for the upcoming week.
- **Air Quality Index (AQI):** Monitor air pollution levels with health recommendations.
- **Search History:** Remembers your recently searched cities for quick access.
- **Unit Switching:** Toggle between Celsius (°C) and Fahrenheit (°F).
- **Shimmer Loading:** Premium skeleton loading effect while fetching data.

## 🛠️ Tech Stack & Libraries
- **State Management:** Flutter Riverpod
- **Networking:** HTTP
- **Location:** Geolocator & Geocoding
- **Storage:** Shared Preferences (for history and last city)
- **Charts:** FL Chart
- **Animations:** Lottie
- **Icons:** Font Awesome & Material Icons
- **Fonts:** Google Fonts (Poppins)
- **UI Effects:** Shimmer

## ⚙️ Setup & Installation
1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/weather_pro_app.git
   ```
2. **Install dependencies:**
   ```bash
   flutter pub get
   ```
3. **Generate App Icons:**
   ```bash
   flutter pub run flutter_launcher_icons
   ```
4. **Run the app:**
   ```bash
   flutter run
   ```
